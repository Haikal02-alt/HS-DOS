# HS-DOS (Haikal Scriptable Disk Operating System)

HS-DOS is an **alternative to MS-DOS**, built on top of Arch Linux.  
It combines the minimalism of DOS-like environments with the flexibility of Arch, offering a scriptable, manual installation flow that mirrors the classic disk operating system.

---

## Features
- **Arch Linux Base**: Rolling release, pacman package manager, Arch repositories.
- **Manual Installation Rituals**:
  - Partitioning with `fdisk`
  - Bootloader setup with GRUB
  - Kernel:`linux-lts`, `linux-zen`
- **DOS-inspired Environment**:
  - Lightweight shell interface
  - Haikal Script runtime for automation
- **Custom Branding**:
  - GRUB theme with HS-DOS identity
  - Nostalgic DOS-like feel
